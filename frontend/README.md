# 🚀 프로젝트 실행 방법

## 1. 개발 환경

- Node.js v18 이상 권장
- npm 사용
- React + TypeScript + Tailwind CSS

## 2. 프로젝트 설치

```
# 저장소 클론
git clone <repository-url>

# 프로젝트 폴더 이동
cd <project-name>

# 패키지 설치
npm install
```

## 3. 개발 서버 실행

```
npm run dev
```

실행 후 브라우저에서 아래 주소에 접속

[http://localhost:5173](http://localhost:5173)

## 4. 참고

실행이 되지 않을 경우 아래 명령어로 재설치 해주세요:

```
rm -rf node_modules package-lock.json
npm install
```
